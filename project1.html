Project 1

조건 입력 기반 여행지 추천 웹사이트

A Travel Destination Recommendation Website Based On Conditional Input

프로젝트 제안 설명

  여행 계획을 세우기 위해 여행 사이트에서 여행지를 검색한 결과, 추천 여행지가 광범위했했다. 뿐만 아니라, 카테고리 란에 '구성원', '인원수', '이동수단' 등 세부 조건이 부족하다는 불편을 느꼈다.

  이 불편을 해소하고자 여행 계획자가 여행을 계획할 때, 조건에 맞는 여행지를 알 수 있도록 <조건 입력 기반 여행지 추천 웹사이트>를 만들고자 한다.


설계도

-입력부


  사용자가 접속한 후 순서대로 행선지, 인원수, 동행인, 일정 난이도, 선호 활동, 이동수단에 대한 선택지를 제시하고 입력 받습니다. 사용자가 선택한 조건들을 JSON 파일로 변환합니다. 변환한 JSON 파일을 모델로 보냅니다.


-추천부


  사용자가 선택한 조건들을 각각의 카테고리에 입력한 후 매핑(Mapping)합니다. 이동수단에 기반하여 매개변수를 설정합니다. Google API를 이용하여 위도/경도 값을 가져옵니다. 여행지 목록을 가져옵니다.여행지에 대한 설명을 가져옵니다. 여행지에 대한 설명을 TF-IDF 벡터화 합니다. 모델을 이용하여 카테고리를 예측합니다. 목표 카테고리별로 여행지를 필터링합니다. 최종 추천 결과를 출력부로 반환합니다.


-출력부


  머신으로부터 API 응답을 받습니다. API를 통해 전달받은 JSON 데이터를 변환합니다. 변환 후 상태(State)를 업데이트합니다. 업데이트한 내용 즉, 추천 여행지를 사용자에게 보여줍니다.


결과

-홈페이지

  홈페이지 화면으로, 사용자가 웹사이트에 접속 시 제일 처음 보이는 화면이다. 'Triplanner'는 이 프로젝트를 제작한 조 명(名)이다.


-회원가입/로그인

  사용자가 웹사이트를 이용하기 위해 회원가입 및 로그인을 하는 화면이다. 회원가입을 위해 이름, 아이디, 비밀번호, 이메일, 핸드폰 번호 등을 입력한다. 회원가입 완료 후, 아이디와 비밀번호를 입력하여 로그인한다. 회원가입 및 로그인 성공 시, 성공 안내 팝업창이 표시된다.

  회원가입 및 로그인 기능은 백엔드와의 API 통신을 이용하여 제작했다.


-제시된 조건 선택

  로그인 후, 행선지 및 조건들을 선택한다. 행선지는 강원도로 한정하였고, 조건을 선택할 수 있는 카테고리로는 인원수, 동행인, 여행 스타일, 선호 활동(중복 선택 가능), 이동수단이 있다. 카테고리별로 조건을 선택한 후, 다음 버튼을 누른다.


-여행지 추천받기

 모델이 조건에 맞는 여행지를 찾을 동안 대기한다. 모델이 반환한 추천 여행지를 보여준다. 모델이 추천해준 관광지, 숙박시설, 음식점 등을 확인한다.


추가 기능

-백엔드와의 API 통신을 이용하여 제작한 기능이다.

-내 일정

  >일정이 없는 경우
  기존에 저장해둔 일정이 없는 경우, 저장된 일정이 없다고 표시되고 아래 버튼을 클릭하여 일정을 만들러 갈 수 있다.

  >일정 저장하기
  여행지 추천 페이지에서 하단에 보면 <일정 저장하기> 버튼이 있다. 이 버튼을 클릭하여 저장할 위치를 선택한다. <내 일정에 저장하기>를 선택한 후, 화면 상단에 '저장되었습니다.'라는 문구를 확인할 수 있다. 내 일정 페이지로 들어가서 저장한 일정을 확인한다.


-그룹

  >그룹 생성, 초대 및 가입
  <그룹 만들기> 버튼을 눌러 그룹을 생성할 수 있고, 가입되어 있는 그룹을 확인할 수 있다. 본인이 가입되어 있는 그룹으로 타사용자를 초대할 수 있다. 타사용자는 자신에게 온 초대 목록을 확인하고, 수락 또는 거절을 할 수 있다. 수락하는 경우, 타사용자 또한 그룹에 가입된다.


  >그룹 일정 저장
  사진과 같이 그룹A, 그룹B, 그룹C에 가입되어 있다고 해보자. 여행지 추천 페이지의 하단에 <일정 저장하기> 버튼을 클릭한다. <내 그룹에 저장하기>를 선택하는 경우, 내가 가입되어 있는 그룹의 이름을 볼 수 있고, 그 중 하나를 선택한다. 저장할 그룹까지 선택하면 화면 상단에 '저장되었습니다.'라는 문구를 확인할 수 있다. 내 그룹 페이지로 가서 저장한 그룹 일정을 확인할 수 있다. 저장한 일정이 없는 경우, '저장된 일정이 없습니다'라는 문구를 확인할 수 있다.


-리뷰

  내 일정 페이지에 저장되어 있는 '내 일정'의 여행지의 <장소 세부정보>를 클릭한다. 작성되어 있는 리뷰가 없는 경우, '작성된 리뷰가 없습니다.'라는 문구를 확인할 수 있다. 작성되어 있는 리뷰가 존재하는 경우, 리뷰(별점 및 내용)를 확인할 수 있다. 리뷰 작성을 원하는 경우, <리뷰 작성> 버튼을 눌러 별점 및 후기를 작성할 수 있다. 작성 후, <저장> 버튼을 누르면 '작성되었습니다'라는 문구와 함께 리뷰가 저장된다. <장소 세부정보> 버튼을 눌러 작성한 리뷰를 확인할 수 있다.
