Project 4

전자 피아노

Electronic Piano

프로젝트 제안 설명

  기초전자회로실험에서 배운 내용을 활용하여 설계 프로젝트를 진행하고자 하였기에, 커패시터와 555타이머를 활용해 주파수에 따라 다른 소리를 내는 피아노를 전자회로로 만들기로 하였다. 칼림바와 같이 작은 크기의 악기도 연주가 가능하다는 것을 보고, PCB보드에 회로를 구성하여 휴대 가능한 피아노를 만들고자 하였다.


동작 설명 및 회로도

-동작설명


  스위치를 누르면 해당 스위치에 연결되어 있는 저항이 555 타이머 회로에 적용된다. 그러면 555 타이머는 불안정 모드로 동작하면서, 저항과 커패시터 값에 의해 결정되는 특정 주파수의 사각파 신호를 지속적으로 생성한다. 생성된 사각파 신호는 출력단을 통해 스피커로 전달되고, 스피커에서 해당 주파수에 대응하는 소리가 발생한다.


-왼손 연주부

  >회로도

  스위치를 누르면 해당 스위치에 연결된 저항이 555 타이머 회로에 적용되어, 특정 주파수의 사각파를 생성하도록 구성되었다. 생성된 신호는 스피커로 전달되어 해당 음의 소리가 발생하도록 한다. 이 회로는 왼손으로 연주하는 음을 출력하기 위한 회로이다.

  >시뮬레이션
  PSpice를 이용하여 각 스위치별로 출력되는 주파수를 측정한 결과이다. 피아노의 음계와 동일한 것을 알 수 있다.


-오른손 연주부

  >회로도

  스위치를 누르면 해당 스위치에 연결된 저항이 555 타이머 회로에 적용되어, 특정 주파수의 사각파를 생성하도록 구성되었다. 생성된 신호는 스피커로 전달되어 해당 음의 소리가 발생하도록 한다. 이 회로는 오른손으로 연주하는 음을 출력하기 위한 회로이다.

  >시뮬레이션
  PSpice를 이용하여 각 스위치별로 출력되는 주파수를 측정한 결과이다. 피아노의 음계와 동일한 것을 알 수 있다.


결과

-결과물 사진

  브레드보드에 실험하여 성공한 후, PCB 보드에 회로를 다시 구성하여 납땜한 사진이다. 위쪽이 왼손 연주부, 아래쪽이 오른손 연주부이며, 외쪽부터 오른쪽 순으로 도 -> 레 -> 미 -> 파 -> 솔 -> 라 -> 시 -> (높은)도 이다.


-결과물 영상


  최종 완성된 전자 피아노를 연주한 영상이다. 왼손 연주부와 오른손 연주부 모두 각 스위치별로 소리가 알맞게 잘 출력되는 것을 확인할 수 있다.
